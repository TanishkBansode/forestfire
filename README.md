# Forest Fire Simulator

- Green dots shown are trees.
- When lightning strucks to the trees, they burn. Yellow animation is shown to tell that they are burning.

It is inspiration from a Veritasium video: https://youtu.be/HBluLfX2F_k?si=GHz5MZxn4oxOsI0r&t=1270, and I have plans to make it as a wallpaper or a screensaver.
